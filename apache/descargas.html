Es esencial que verifique la integridad del archivo descargado utilizando la firma ( .ascarchivo) de PGP o un hash ( .md5o .sha*archivo). Lea la sección Verificación de las versiones de Apache Software Foundation para obtener más información sobre por qué debe verificar nuestras versiones. La firma PGP puede ser verificada usando PGP o GPG. Primero descargue el archivo de firma y KEYSel de ascfirma para la distribución correspondiente. Asegúrese de obtener estos archivos desde el sitio de distribución principal, en lugar de hacerlo desde un espejo. Luego verifique las firmas usando

% gpg --import KEYS % gpg --verify download_file.asc download_file

o

% pgpk -a TECLAS % pgpv descargado_archivo.asc

o

% pgp -ka KEYS % pgp descargado_file.asc

o

% gpg --print-md SHA1 download_file
