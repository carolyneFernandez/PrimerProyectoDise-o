Instalando NGINX Plus en Debian y Ubuntu

  • Debian 8 ("Jessie")
  • Debian 9 ("estirar")
  • Ubuntu 14.04 LTS ("Trusty") (i386, x86_64, aarch64)
  • Ubuntu 16.04 LTS ("Xenial") (i386, x86_64, ppc64le, aarch64)
  • Ubuntu 18.04 ("Bionic")

Para instalar NGINX Plus en Debian o Ubuntu:

  1. Cree el directorio / etc / ssl / nginx : sudo mkdir / etc / ssl / nginx cd / etc / ssl / nginx
  2. Inicie sesión en el Portal del cliente NGINX Plus y descargue los archivos nginx-repo.crt y nginx-repo.key .
  3. Copie los archivos al directorio / etc / ssl / nginx / : sudo cp nginx-repo.crt / etc / ssl / nginx / sudo cp nginx-repo.key / etc / ssl / nginx /
  4. Descargue la clave de firma NGINX de nginx.org y agréguela: sudo pkg install ca_root_nss
  5. Instale el paquete apt-utils y el repositorio NGINX Plus. sudo wget https://nginx.org/keys/nginx_signing.key sudo apt-key add nginx_signing.key
    • Para debian sudo apt-get install apt-transport-https lsb-release-certificados ca printf "deb https://plus-pkgs.nginx.com/debian` lsb_release -cs` nginx-plus \ n "| sudo tee /etc/apt/sources.list.d/nginx-plus.list
    • Para Ubuntu sudo apt-get install apt-transport-https lsb-release-certificados ca printf "deb https://plus-pkgs.nginx.com/ubuntu` lsb_release -cs` nginx-plus \ n "| sudo tee /etc/apt/sources.list.d/nginx-plus.list
  6. Descargue el archivo 90nginx en /etc/apt/apt.conf.d :sudo wget -q -O /etc/apt/apt.conf.d/90nginx https://cs.nginx.com/static/files/90nginx
  7. Actualice la información del repositorio:sudo apt-get update
  8. Instale el paquete nginx-plus . Cualquier paquete anterior de NGINX Plus se reemplaza automáticamente.sudo apt-get install -y nginx-plus
